# Azure RBAC & IAM

## Overview

This project demonstrates Identity and Access Management (IAM) and Role-Based Access Control (RBAC) in Microsoft Azure.

## Services Used

- Microsoft Entra ID
- Security Groups
- Azure RBAC
- Resource Groups
- Terraform

## Features

- User Management
- Security Group Management
- Role Assignments
- Least Privilege Access Control

## RBAC Flow

User → Group → Role → Resource Group

## Role Used

Contributor

## Benefits

- Centralized access management
- Improved security
- Scalable permissions model
- Enterprise governance
